Description:
BUMPIN' Hockey! is a 2-player hockey game played with bumper cars. A game consists of three 2-minute periods. The playing area is reset at the end
of each period and play is resumed on next click. A winner or tie is determined at the end of the game. The minimal game plays "Countin' Bumps". 
The object of this game is simply to have the most number of bumps at the end of the game.

Controls:

Player 1

W = accelerate
S = backup
A = left
D = right

Player 2

Up arrow = accelerate
Down arrow = backup
Right arrow = left
Right arrow = right
